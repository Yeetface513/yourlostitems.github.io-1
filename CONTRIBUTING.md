# How to contribute

- Create a useful and well made pull request.
- wait for it to be discussed and potentially accepted.
- To assist in creating this helpful pull request, we have included a diagram.

```mermaid
graph TD
A[Suffering];
B[Completion];
C[Tutorial hell];
D[Procrastination];
E[Hard work];
F[Repeat];
A-->C;
C-->D;
D-->E;
E-->F;
F-->A;
F-->B;
```
